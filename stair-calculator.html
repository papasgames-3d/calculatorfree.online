Stair Construction Tips

📐 Rise & Run Formula

Ideal ratio: 2 x Rise + Run = 24-25 inches. Common comfortable ratio is 7" rise with 11" run.

📏 Consistent Dimensions

All rises should be identical within 3/8". Variation can cause tripping hazards.

🛠️ Stringer Placement

Use 2x12 lumber for stringers. Space stringers max 16" on center for 5/4" treads.

🔧 Secure Attachment

Attach stringers with joist hangers or ledger board. Use appropriate fasteners for load.

🚪 Headroom Check

Minimum 6'8" headroom throughout stair run. Check local codes for requirements.

🛡️ Safety Features

Install handrails 34-38" high. Guardrails required for stairs over 30" high.
